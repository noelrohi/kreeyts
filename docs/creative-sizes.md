# Creative Sizes

Supported video and image sizes with their aspect ratios. Canonical machine-readable specs live in `packages/product/src/placements.ts`.

## Video sizes

| Size (px)   | Aspect ratio    |
| ----------- | --------------- |
| 1280 × 720  | 16:9 (≈ 1.78:1) |
| 720 × 1280  | 9:16 (≈ 0.56:1) |
| 1080 × 1080 | 1:1             |
| 300 × 250   | 6:5 (≈ 1.20:1)  |
